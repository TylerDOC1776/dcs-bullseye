DCS World Server Platform

DCS Bullseye is a Discord-native management platform for DCS World dedicated servers. Start missions, manage players, and monitor your server — all without touching the machine.

View on GitHub

For Players

Everything you need, right in Discord

Check server status, see active missions, and get notified about what's happening — without leaving your voice channel.

📡

Live Server Status

See which servers are online, what mission is running, and how many players are connected in real time.

🗺️

Mission Info

Know what's loaded before you hop in. Mission name, theater, and player count — all from a single command.

📊

Player Stats

Register your DCS pilot name and track your session history, flight time, and activity across all connected servers.

🔔

Server Alerts

Automatic Discord notifications when servers start, stop, crash, or change mission.

For Community Hosts

Run your server. Control it from Discord.

Bullseye gives server operators full remote control over DCS dedicated servers through a lightweight Windows agent and a centralized orchestrator.

⚙️

Full Server Control

Start, stop, restart, and update DCS — all through Discord slash commands with role-based access control.

🗃️

Mission Management

Upload, load, download, and rotate missions remotely. No RDP required.

🔐

RBAC Permissions

Three permission tiers — viewer, operator, admin — so you control who can do what on your server.

📦

One-Command Install

A single PowerShell command installs and registers the agent. No manual config files, no port forwarding required.

🔄

Auto DCS Updates

Trigger DCS World updates remotely with live Discord progress tracking while the updater runs.

🛡️

No Public IP Needed

The agent connects out to the platform via an encrypted reverse tunnel. Your Windows machine never needs to be publicly exposed.
